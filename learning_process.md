# Installing

* Node: `sudo apt-get install nodejs:18/common``;
* npm packages: 
   * npm install react-bootstrap bootstrap
   * npm install react-bootstrap
   * npm install react-router-dom react-router-bootstrap

# Project Setup
After installed run
```
npx create-react-app MY_APP_NAME
```
